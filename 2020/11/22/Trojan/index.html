连接 VPS

  1. 如果有必要，重装 VPS 系统。

  2. 取得 IP 地址Public IP address和端口号SSH Port并记录密码。

  3. Windows 连接 VPS 需要下载 Xshell 或 PuTTY 。

  4. 以 Xshell 为例，File|New：

    Name随便写；

    Protocol默认SSH；

    Host填入 IP 地址；

    Port Number填入端口号。

    点击确认，在弹出窗口中输入用户名和密码，用户名默认为root。若没有出现弹窗，说明 VPS 被墙。
